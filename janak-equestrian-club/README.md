# Janak Equestrian Club

A premium 3D-inspired website built with Next.js, Tailwind CSS, React Three Fiber, Drei, and Framer Motion.

## Design Direction
This project is based on a cinematic purple-dusk visual language:
- layered gradients
- collectible card style presentation
- immersive glass panels
- soft gold highlights
- lightweight 3D hero scenes

## Tech Stack
- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber
- Drei
- Three.js

## Features
- Premium landing hero with animated 3D scene
- About section with abstract rotating 3D ring scene
- Program cards for beginner, intermediate, and elite riders
- Interactive featured horse selector
- Facilities and event showcase
- Membership enquiry form with working API route
- Responsive layout with luxury dark theme

## Run Locally

```bash
cd janak-equestrian-club
npm install
npm run dev
```

Open:
- Website: http://localhost:3000

## API Routes
- `POST /api/membership`
- `GET /api/events`
- `GET /api/horses`

## Notes
The 3D scenes use stylized primitives instead of heavy external models so the project runs immediately after install.
